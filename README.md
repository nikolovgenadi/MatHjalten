is a full-stack Progressive Web Application (PWA) built to reduce food waste by letting users share surplus food instead of throwing it away.
Users can register, log in, create ads for food/drinks, upload images, browse and filter available items, reserve offers, and manage their own ads.
The UI is mobile-first and the app can be installed directly from the browser as a PWA.

# Tech Stack

# Frontend

React 19 (Vite)

React Context API (Auth & Ads state)

React Router (view handling/navigation)

Tailwind CSS (v4, mobile-first styling)

Lucide React (icons)

# Backend

Node.js + Express 5

MongoDB Atlas + Mongoose

Zod (request validation)

Multer (image uploads)

bcryptjs (password hashing)

jsonwebtoken (JWT auth)

cookie-parser (HTTP-only auth cookies)

# Other

PWA manifest (installable app)

Render (hosting)

Thunder Client (API testing during development)
